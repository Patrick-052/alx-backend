# alx-backend

## 0x01-caching
This is the caching project for the ALX backend curriculum.

## Learning Objectives
- Meaning of a caching system
- Meaning of FIFO and LIFO
- Meaning of LRU and MRU
- Meaning of LFU
- Purpose and limits of a caching system
